1.) Open "file-explorer (Datei-Explorer)" and navigate to the directory where the file "asciiChristmasTree.class" is and copy the path.
2.) Press the windows key and type "cmd" and press return.
3.) Type the proper drive letter where the file is (mentioned in step 1) and add a ":" to it and press return.
4.) Type "cd", then press space, then paste the copied path (mentioned in step 1) and press return.
5.) Type "java asciiChristmasTree", then press space, then type in the height you like as a number, then press space, then type in "yes" if you want a star on top of the tree or type anything else but do not let it empty for having no star on it, then press return. Notice that you write "yes" correctly in small letters to make it working!
